# MediaSuite
NAS 媒体服务一站式管理面板

---

## ✨ 功能亮点
- ✅ 一键部署：MoviePilot / Emby / qBittorrent / CloudDrive 等全套媒体服务
- ✅ 可视化交互菜单，支持单服务启停 / 重建 / 卸载
- ✅ 配置自动备份与一键还原
- ✅ 全平台 Linux 通用（Ubuntu / Debian / 群晖 / 威联通）
